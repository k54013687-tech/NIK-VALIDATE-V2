### About Tool

</div>

NIK Validate Tools adalah alat validasi struktur dan konsistensi Nomor Induk Kependudukan (NIK) Indonesia (16 digit). Tools ini dibuat untuk keperluan edukasi, input validation, dan pengecekan format, bukan untuk verifikasi data kependudukan resmi.

> ⚠️ Peringatan Penting
Tools ini tidak terhubung ke database Dukcapil dan tidak dapat memastikan keabsahan identitas seseorang. Hanya instansi berwenang yang dapat melakukan verifikasi NIK secara resmi.
